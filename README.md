# cursor_html_css_clone
# Cursor Landing Page Clone

A desktop-first **landing page clone** inspired by the Cursor AI website.  
This project focuses on recreating the visual layout and structure using **HTML and CSS**.

---

## Overview

This project replicates a modern developer tool landing page similar to Cursor.  
The goal is to practice frontend development skills including layout design, styling, and UI structuring.

---

## Tech Stack

- HTML5
- CSS3
- Flexbox
- Responsive Layout Basics

---

## Features

- ✅ Navigation Bar
- ✅ Hero Section
- ✅ Brand Showcase
- ✅ Testimonials Section
- ✅ Feature Cards
- ✅ Modern Footer Design
- ✅ Desktop-first Layout
